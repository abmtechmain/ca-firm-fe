Gallery Preview — add your photos here

1. Copy JPG or PNG files into this folder (e.g. event-2026.jpg).
2. Open app/constants/index.ts and find GALLERY_PREVIEW_IMAGES.
3. Add a line:
   { id: '7', src: '/images/gallery/event-2026.jpg', alt: 'Short description', caption: 'Optional label' }

Refresh the Gallery Preview page to see new images.
